Cada versión de Odoo tiene su propio vault nombrado `vault{version}` dentro de su carpeta:
- `/home/axel/odoo/16/vault16/`
- `/home/axel/odoo/17/vault17/`
- `/home/axel/odoo/18/vault18/`
- `/home/axel/odoo/19/vault19/`

**Why:** El usuario quiere vaults separados por versión de Odoo para mantener el contexto aislado. El nombre incluye la versión para identificarlo fácilmente desde Obsidian.

**How to apply:** Usar siempre el vault de la versión activa (según el directorio de trabajo). Consultar /Modulos/, /Bugs/ y /Diario/ para contexto histórico antes de trabajar.
