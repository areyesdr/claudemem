Skill `odoo-sv-dev` creada en `~/.claude/skills/odoo-sv-dev/SKILL.md`.

**Why:** El repositorio l10n_sv_edi tiene 157+ módulos con patrones muy específicos. La skill captura los patrones, convenciones y flujos DTE para no tener que re-explorar el código en cada sesión.

**How to apply:** Usar `/odoo-sv-dev` al iniciar trabajo en módulos Odoo. La skill referencia las rutas exactas del código fuente para que Claude pueda leer antes de escribir.

Basada en: l10n_sv_edi v17.10.29, autor TierraNube - Intelitecsa (Pedro Arroyo, Nestor Ulloa, Francisco Trejo, Axel Reyes).
